Enhance Network Security with AI-Powered Honeypot

This project aimed to combine the passive surveillance strength of honeypots with the intelligence of AI-based detection and prevention models to create a self-defending security system.

Our goal was to address the lack of automated, adaptive security mechanisms in existing honeypot deployments. By simulating a real world enterprise network within a virtualized EVE- NG environment, we were able to integrate Cowrie and Dionaea honeypots, simulate cyberattacks using Kali/Parrot OS, and design an AI model that classifies, responds, and prevents threats in real-time.
